# Smart Therapy Tracking System for Autism Patients   
## -Jointly created by 4 students (cyik, Ilhaam, Heyi and Haoze) in March 2026 for Capstone Project.  

# What is EZTherapy?
EZTherapy is a web-based smart therapy tracking system for autism patients that is specifically designed to assist therapists more understand their patients and manage therapy appointment schedules as well as provide meaningful insights into therapy data and progress, while also bridging the communication gap between the therapist and patient, it also has a AQ10 and AI Chatbot feature that allows the patient to be more understood. It uses frameworks such as React.js for the frontend user interface, making it responsive and allowing for cross platform support in the future. It uses SQLite and SQLAlchemy for the database and user authentication to store user profiles and preferences. And it uses face-api.js to handle all machine learning models and algorithms.

The platform was specifically designed to provide features such as a centralized dashboard with visual analytics of therapy data, therapy appointment scheduling, behavioral tagging for patients through AQ10 questionnaire, facial emotion detection tracking to help therapists assist their autism diagnosis, and an AI Chatbot LLM to allow patients to ask questions. The frontend for the application is designed to have a minimalistic and easy to use user interface which makes it easy for therapist and patient to navigate. Other than that, the security for this application is also encrypted and highly secure, it uses advanced encryption methods such as hashing methods which helps to store the user’s passwords in a hashed database.

Through intelligent analysis and complete industry standard features that help therapists streamline their workflow, EZTherapy aims to make therapist’s jobs easier, while also improving the quality of their therapy sessions through our smart analytics, which ultimately help autistic patients to get the therapy support that they need.
